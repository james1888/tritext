# tritext

Make triangle text from your commandline 🙈📐

![](https://media.giphy.com/media/klzxzIdo8MK39QM9JP/giphy.gif)

## Installation

```bash
npm install -g tritext-cli
```

## Usage

```bash
> tritext "Hello world"

= = = = = = = = = =

H E L L O W O
H E L L O W

= = = = = = = = = =

Copied to clipboard!
```
